Run + Yoga

No gym needed. Running plus daily yoga and crunches — four ability levels inside, from first 10K to advanced half marathon.

Track 2

Run + Yoga + Gym

Same running ladder, with gym riding alongside every day — lighter on long-run days, fuller on shorter ones.

Track 3

Team White

For new starters with gym access who've never actually been. Run-majority, light full-body & core gym 3x/week, and two days a week let you swap the run for a cycle instead.

Track 4

Team Black

For gym-track runners who also have cycling access but aren't doing a daily run+gym habit yet — cycling built in properly, once or twice a week, instead of a suggestion to swap on your own.

Track 5

Team Red

Same daily running + daily gym hybrid as Team Saffron, but for those without cycling access — every cycle slot becomes a run instead.

Track 6

Team Saffron

Daily running and daily gym, with cycling built in 2–3 times a week, not just as an option.
